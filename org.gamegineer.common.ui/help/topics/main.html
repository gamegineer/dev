Using the help system

Browse topics in the Contents tab on the left. Click on a topic to have it displayed. Use the Back and Forward buttons to navigate within the history of viewed topics.

Index

To show an alphabetically sorted index of keywords, select the Index tab. Not all products contain an index.

Search

To locate topics on a particular subject in the documentation, select the Search tab.

Favorites

To add and remove favorite topics, select the Favorites tab.
