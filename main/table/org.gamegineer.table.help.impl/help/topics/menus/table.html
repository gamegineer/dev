Table menu

The Table menu commands are described below.

Name Description
Add
Adds a new component to the table. The available components are described below.
Card
Adds a new card to the selected container. Choose the card suit (Clubs, Diamonds, Hearts, Spades, or Special) followed by the card rank (Ace, Two, Three, Four, Five, Six, Seven, Eight, Nine, Ten, Jack, Queen, or King for the Clubs, Diamonds, Hearts, and Spades suit; Joker for the Special suit).
Card Pile
Adds a new card pile to the table.
Deck
Adds a new deck of cards to the selected container. Choose the type of deck to add.
Flip Flips the selected component.
Layout
Changes the layout of the selected container. The available layouts are described below.
Accordian Down
Lays out the components one on top of the other with each component offset slightly below the one under it.
Accordian Left
Lays out the components one on top of the other with each component offset slightly to the left of the one under it.
Accordian Right
Lays out the components one on top of the other with each component offset slightly to the right of the one under it.
Accordian Up
Lays out the components one on top of the other with each component offset slightly above the one under it.
Stacked
Stacks the components one on top of the other. This is the default layout.
Remove Removes the selected component.
Remove All Removes all components from the selected container.
